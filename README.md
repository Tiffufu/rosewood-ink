# Rosewood Ink

Rosewood Ink is a warm, parchment-inspired theme for **Obsidian** designed for long-form writing.

The theme softens Obsidian’s interface into a paper-like workspace: muted backgrounds, ink-like text contrast, and restrained editorial accents. The goal is to create a calm drafting environment that remains readable during extended writing sessions without feeling visually flat.

Light mode uses warm parchment tones with terracotta markup accents.  
Dark mode preserves the same ink-and-paper aesthetic for writing in low-light environments.

Rosewood Ink favors **clarity and writing comfort over heavy interface customization**.

---

## Features

- parchment-inspired light palette
- matching dark mode tuned for readability
- mulberry accent inspired by fountain pen inks
- reduced visual noise for drafting and revision
- consistent typography and spacing for long writing sessions

---

## Installation

Clone or download this repository and place the theme files in your vault:

```text
.obsidian/themes/rosewood-ink/
```

Then enable **Rosewood Ink** in:

Settings → Appearance → Themes
